# Магазин аксессуаров для домашних животных

Лабораторная работа 16 (2 семестр) по Django. Приложение для интернет-магазина товаров для домашних питомцев.

## Страницы магазина

| URL | Название | Описание |
|-----|----------|----------|
| `/` | Главная | Приветственная страница с навигацией по сайту |
| `/about/` | Об авторе | Информация о разработчике сайта |
| `/shop/` | О магазине | Описание тематики магазина и ассортимента товаров |

## Установка и запуск

1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать окружение: `venv\Scripts\activate`
4. Установить зависимости: `pip install django`
5. Запустить сервер: `python manage.py runserver`

## Зависимости

- Python 3.14
- Django (последняя версия)
- SQLite
